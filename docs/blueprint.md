# **App Name**: CodeVerse

## Core Features:

- Создание постов: Пользователи могут создавать посты с текстом, изображениями и фрагментами кода, отмеченные соответствующими темами и пользовательскими тегами.
- Тематическая категоризация: Посты автоматически распределяются по предопределенным темам (Новости, Программирование, Дизайн и т.д.) для улучшения обнаружения контента. Пользователь может выбрать предопределенную тему при создании поста.
- Форум вопросов: Раздел, где пользователи могут задавать вопросы и получать ответы от сообщества.
- Фильтрация и сортировка: Пользователи могут фильтровать и сортировать посты по категориям, тегам, дате и популярности (лайки, просмотры).
- Аутентификация пользователя: Безопасная система регистрации и входа в систему с управлением профилем.
- Предложение тегов с помощью ИИ: В качестве инструмента, предлагайте релевантные теги на основе содержания поста, используя генеративный ИИ.

## Style Guidelines:

- Основной цвет: Спокойный синий (#64B5F6), чтобы вызвать доверие и спокойствие.
- Цвет фона: Очень светло-голубой (#E3F2FD), близкий к белому, для ощущения легкости и воздушности.
- Цвет акцента: Мягкий фиолетовый (#BA68C8), чтобы добавить нотку футуризма и креативности.
- Шрифт для тела и заголовков: 'Inter' (sans-serif) для современного и удобочитаемого интерфейса.
- Используйте чистые, минималистичные иконки для навигации и действий.
- Просторный и чистый макет для обеспечения удобочитаемости и ощущения открытости.
- Тонкие переходы и анимации для улучшения пользовательского опыта, не отвлекая.
